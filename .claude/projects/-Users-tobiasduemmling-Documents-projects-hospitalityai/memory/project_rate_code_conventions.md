In Mews rate/voucher codes, `VP` stands for **Vollpension** (full board).

**Why:** Tobi clarified this so code comments, docs, and mappings use the correct term.
**How to apply:** When explaining, documenting, or extending rate codes (e.g. in rate_plans.py or promocodes.md), refer to VP as Vollpension.
